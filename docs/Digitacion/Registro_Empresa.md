# Registro de Empresa

Para registrar una nueva empresa en la plataforma InvoiCy, el usuario deberá acceder al **Panel de Control** y, en el menú **"Gestión"**, seleccionar la opción **"Nueva empresa"**, tal como se muestra en la siguiente imagen.
