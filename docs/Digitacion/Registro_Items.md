# Registro de Ítems

Para registrar un nuevo ítem en la plataforma InvoiCy, el usuario deberá acceder al **Panel de Control** y, en el menú **"Digitación"**, seleccionar la opción **"Ítems de la digitación"**, tal como se muestra en la siguiente imagen.
