# Digitación y emisión de documentos

Para emitir un Documento Tributario Electrónico (DTE) a través de la pantalla de Digitación, el usuario debe acceder al sistema InvoiCy y, en el menú lateral izquierdo, seleccionar la opción **"Emitir"**.

<figure markdown="span" style="max-width: 600px; margin: 0 auto; text-align: center;">
  ![Image title](../img/Artigos/Digitacion_Emision_Figura.png){ loading=lazy style="max-width: 100%; height: auto;" }
  <figcaption>Opción "Emitir", ubicada en el menú lateral izquierdo</figcaption>
</figure>

Al seleccionar esta opción, el usuario será dirigido al primer paso del proceso de digitación, donde deberá elegir el tipo de documento que desea emitir.
